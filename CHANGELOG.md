# Changelog

### Added
- Initial release with 18 skills across 4 plugins

- **analysis-skills**: 4 analysis workflow skills for user profiling, tweet analysis, user comparison, and KOL influence reports
- **research-skills**: 4 research workflow skills for topic research, trend monitoring, community and list exploration
- **utility-skills**: 2 utility skills for quick user lookup and job search
